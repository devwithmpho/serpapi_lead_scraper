# Serpapi Lead Scraper

This tool uses the serpapi library to scrape data from places on Google Maps.
It fetches the data using the API, cleans it, and puts it into a CSV file.

The reason why I built this is because I am running a business and need a quick automated method for finding leads.
With this tool, the only thing I do manually is filter by the best leads.

To run the tool, run:

```bash
streamlit run [project_location]/app.py
```
